# Serietv  - Bot di Telegram per la gestione delle serie tv film e altro

SerieTvBot è un codice Python creato utilizzando la libreria pyTelegramBotAPI e SQLAlchemy come ORM su un database SQLite3 per la gestione dei file su Telegram. Tieni presente che questo codice è solo a scopo didattico non sei obbligato a usarlo

## Alcune funzionalità del bot includono:

- Spazio di archiviazione cloud illimitato su Telegram per l'archiviazione dei file
- Backup automatici su un canale Telegram
- Divertente gestione della chat ispirata a Dragonball (sfere del drago, fagioli e cattivi appariranno nella chat di Telegram)

Ricordo che per garantire il corretto funzionamento, configurare il file settings.py inserendo il token del bot e, facoltativamente, il token del bot di prova impostando il flag TEST su 1.
